What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes the how HTML elements are to be displayed on screen, paper, or their in media.
  3. CSS is not necessaryCSS saves a lot work.it can control the layputs of multiple web pages all at once.
  4. External stylesheets are stored in CSS Styles.

CSS can be to added HTML documents in 3 ways

  • Inline- by using the style attribute in HTML elements1
  • Internal - by using a style element in a head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS Class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an elements has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors availabe. Two of the most commonly used are class and ID. Both are used to get elements to which style should be applied.
